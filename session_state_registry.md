# DIG Analytics ‚Äî Session State Registry

| `st.session_state.*` | What it stores | Initialized by | Read by (LLM agent) |
|---|---|---|---|
| `initialized` | Boot flag ‚Äî prevents re-init on rerun | app.py startup | None |
| `stage` | Current pipeline stage string (START ‚Üí AUDIT ‚Üí RESEARCH ‚Üí ANALYSIS ‚Üí DASHBOARD) | app.py startup (`"START"`) | PM Agent (as `CURRENT_STAGE`) |
| `raw_data` | Uploaded CSV as pandas DataFrame | app.py ‚Äî after `pd.read_csv()` | None ‚Äî LLM never sees raw data |
| `metadata` | Profiler output dict ‚Äî schema, nulls, outliers, distributions | Stage 4 ‚Äî `profiler.py` | DE Agent, Researcher Agent, PM Agent |
| `pm_summary` | Last PM Agent user_message (overwritten each gate call) | PM gate calls | Rendered in PM SUMMARY tab |
| `pm_summaries` | Accumulated list of all PM user_messages across AUDIT + RESEARCH gates | app.py startup (`[]`) | Rendered in PM LOG tab |
| `pm_ready` | Bool ‚Äî whether PM Agent approved stage transition | PM gate calls | AUDIT stage ‚Äî gates PROCEED_TO_RESEARCH button |
| `report_view` | Active tab in Living Report panel | app.py startup (`"metadata"`) | app.py ‚Äî content renderer |
| `de_findings` | DE Agent output ‚Äî quality score, issues, outliers JSON | Stage 8 ‚Äî DE Agent call | PM Agent, Researcher Agent |
| `research_paths` | 5 research path options generated by Researcher Agent | Stage 9 ‚Äî Researcher Agent call | PM Agent (to present to user) |
| `current_path` | The single research path the user selected | app.py ‚Äî user selection UI | DA/Stats Agent, PM Agent, Switchboard |
| `tool_result` | Raw output dict from the Starter Kit function | Stage 6 ‚Äî `switchboard.py` | DA/Stats Agent |
| `da_findings` | DA/Stats Agent interpretation of tool result | Stage 10 ‚Äî DA Agent call | Synthesis Engine, BI Developer Agent, PM Agent |
| `analysis_results` | List of all completed path findings (accumulates across loop) | app.py startup (`[]`) | Synthesis Engine, PM Agent |
| `master_report` | Living markdown report ‚Äî rewritten after each path | app.py startup (placeholder string) | Synthesis Engine (rewrites it), rendered in col_report |
| `chart_configs` | Plotly chart config JSON from BI Developer Agent | Stage 11 ‚Äî BI Developer Agent call | app.py ‚Äî `st.plotly_chart()` |
| `history_logs` | List of timestamped log dicts for the terminal panel | app.py startup (one system entry) | None ‚Äî display only |

## Sub-Attributes

### `metadata`
Produced by `profiler.py` ‚Äî `get_dataset_profile()`

| Key | Type | Description |
|---|---|---|
| `shape` | dict | `{"rows": int, "cols": int}` |
| `full_row_count` | int | Total rows before any sampling |
| `duplicate_rows` | int | Count of exact duplicate rows |
| `columns` | list[dict] | One entry per column ‚Äî see below |
| `numeric_summary` | dict | Keyed by column name ‚Äî see below |
| `categorical_summary` | dict | Keyed by column name ‚Äî see below |
| `datetime_flags` | dict | Keyed by column name ‚Äî bool, whether column parses as datetime |

**`columns` entry:**
| Key | Type | Description |
|---|---|---|
| `name` | str | Column name |
| `dtype` | str | Pandas dtype string |
| `null_count` | int | Number of null values |
| `null_pct` | float | Null percentage |
| `sample_values` | list[str] | Up to 5 unique values |

**`numeric_summary` entry (per column):**
| Key | Type | Description |
|---|---|---|
| `mean` | float | Column mean |
| `std` | float | Standard deviation |
| `min` | float | Minimum value |
| `max` | float | Maximum value |
| `outliers_iqr_count` | int | Count of IQR outliers |
| `outliers_zscore_count` | int | Count of Z-score outliers |
| `outlier_sample_values` | list[float] | Up to 5 outlier values |

**`categorical_summary` entry (per column):**
| Key | Type | Description |
|---|---|---|
| `unique_count` | int | Number of distinct values |
| `top_values` | dict | Top 5 values with counts `{value: count}` |

---

### `de_findings`
Produced by DE Agent (Stage 8) ‚Äî validated by Pydantic

| Key | Type | Description |
|---|---|---|
| `quality_score` | int | 1‚Äì10 rating |
| `quality_score_reason` | str | One sentence explanation |
| `dataset_summary` | dict | `{total_rows, total_columns, is_sample}` |
| `columns` | list[dict] | `{name, type, sample_value}` per column |
| `quality_issues` | list[dict] | `{issue, detail, affected_column}` ‚Äî empty if none |
| `outliers` | list[dict] | `{column, value, reason}` ‚Äî empty if none |

---

### `history_logs` entry
| Key | Type | Description |
|---|---|---|
| `time` | str | Timestamp `HH:MM:SS` |
| `msg` | str | Log message |
| `type` | str | `"system"`, `"info"`, or `"error"` |

---

### `analysis_results` entry
Accumulates after each completed research path (Stage 13 loop)

| Key | Type | Description |
|---|---|---|
| `path` | dict | The selected research path |
| `tool_result` | dict | Raw Starter Kit output |
| `da_findings` | dict | DA Agent interpretation |
| `chart_config` | dict | BI Developer Plotly config |
