# DIG Analytics — Session State Registry

| `st.session_state.*` | What it stores | Initialized by | Read by (LLM agent) |
|---|---|---|---|
| `initialized` | Boot flag — prevents re-init on rerun | app.py startup | None |
| `stage` | Current pipeline stage string (`START → AUDIT → RESEARCH → ANALYSIS → DASHBOARD`) | app.py startup (`"START"`) | PM Agent (as `CURRENT_STAGE`) |
| `raw_data` | Uploaded CSV as pandas DataFrame | app.py — after `pd.read_csv()` | None — LLM never sees raw data |
| `metadata` | Profiler output dict — schema, nulls, outliers, distributions | app.py — after `get_dataset_profile()` | DE Agent, Researcher Agent, PM Agent |
| `de_findings` | DE Agent output — quality score, issues, outliers JSON | AUDIT stage — DE Agent call | PM Agent, Researcher Agent |
| `de_running` | Bool — whether DE Agent is currently executing (spinner lock) | app.py startup (`False`) | None — internal UI flag |
| `pm_ready` | Bool — whether PM Agent approved stage transition out of AUDIT | PM gate call at AUDIT | AUDIT stage — gates PROCEED button |
| `pm_summaries` | Accumulated list of all PM `user_message` strings across all gate calls | app.py startup (`[]`) | None — rendered in PM SUMMARY / PM LOG tabs |
| `pm_final_summary` | PM Agent final synthesis string produced at DASHBOARD entry | DASHBOARD stage — PM gate call | BI Agent (as `pm_final_summary` param) |
| `report_view` | Active tab key in Living Report panel | app.py startup (`"metadata"`) | None — app.py content router only |
| `history_logs` | List of timestamped log dicts for the terminal panel | app.py startup (one system entry) | None — display only |
| `master_report` | Legacy living markdown placeholder — fallback render if no view matches | app.py startup (placeholder string) | None |
| `research_paths` | 5 research path options generated by Researcher Agent | AUDIT stage — Researcher Agent call | None — user selects from UI |
| `primary_metric` | Dict identifying the primary binary KPI column, or `None` if not detected | AUDIT stage — Researcher Agent call | None — used in HTML report builder |
| `current_path` | The single research path the user selected | RESEARCH stage — user click | DA Agent (as `path`), PM Agent (as `selected_path`) |
| `tool_result` | List of raw output dicts from Starter Kit functions (one per tool instruction) | ANALYSIS stage — Switchboard execution | DA Agent (as `tool_results`) |
| `da_findings` | DA Agent interpretation of tool results for the current path | ANALYSIS stage — DA Agent call | None — reset to `None` on next path; persisted in `analysis_results` |
| `analysis_results` | List of all completed path dicts — accumulates across the loop | app.py startup (`[]`) | PM Agent (as `previous_findings`), BI Agent, Synthesis Agent |
| `chart_configs` | BI Developer Agent output — KPIs + Plotly chart configs + narrative | DASHBOARD stage — BI Agent call | Synthesis Agent (as `chart_configs`) |
| `synthesis` | Synthesis Agent output — cross-path narrative + recommendations | DASHBOARD stage — Synthesis Agent call | None — rendered in DASHBOARD and HTML report |
| `api_call_count` | Running count of LLM API calls made this session | app.py startup (`0`) | None — footer display only |
| `total_input_tokens` | Cumulative input tokens across all API calls | app.py startup (`0`) | None — footer display only |
| `total_output_tokens` | Cumulative output tokens across all API calls | app.py startup (`0`) | None — footer display only |
| `estimated_cost_usd` | Cumulative estimated USD cost of all API calls | app.py startup (`0.0`) | None — footer display only |

## Sub-Attributes

### `metadata`
Produced by `profiler.py` — `get_dataset_profile()`

| Key | Type | Description |
|---|---|---|
| `shape` | dict | `{"rows": int, "cols": int}` |
| `full_row_count` | int | Total rows before any sampling |
| `duplicate_rows` | int | Count of exact duplicate rows |
| `columns` | list[dict] | One entry per column — see below |
| `numeric_summary` | dict | Keyed by column name — see below |
| `categorical_summary` | dict | Keyed by column name — see below |
| `datetime_flags` | dict | Keyed by column name — bool, whether column parses as datetime |

**`columns` entry:**
| Key | Type | Description |
|---|---|---|
| `name` | str | Column name |
| `dtype` | str | Pandas dtype string |
| `null_count` | int | Number of null values |
| `null_pct` | float | Null percentage |
| `sample_values` | list[str] | Up to 5 unique values |

**`numeric_summary` entry (per column):**
| Key | Type | Description |
|---|---|---|
| `mean` | float | Column mean |
| `std` | float | Standard deviation |
| `min` | float | Minimum value |
| `max` | float | Maximum value |
| `outliers_iqr_count` | int | Count of IQR outliers |
| `outliers_zscore_count` | int | Count of Z-score outliers |
| `outlier_sample_values` | list[float] | Up to 5 outlier values |

**`categorical_summary` entry (per column):**
| Key | Type | Description |
|---|---|---|
| `unique_count` | int | Number of distinct values |
| `top_values` | dict | Top 5 values with counts `{value: count}` |

---

### `de_findings`
Produced by DE Agent — validated by Pydantic

| Key | Type | Description |
|---|---|---|
| `quality_score` | int | 1–10 rating |
| `quality_score_reason` | str | One sentence explanation |
| `dataset_summary` | dict | `{total_rows, total_columns, is_sample}` |
| `columns` | list[dict] | `{name, type, sample_value}` per column |
| `quality_issues` | list[dict] | `{issue, detail, affected_column}` — empty if none |
| `outliers` | list[dict] | `{column, value, reason}` — empty if none |

---

### `primary_metric`
Produced by Researcher Agent — `None` if no binary column detected

| Key | Type | Description |
|---|---|---|
| `label` | str | Human-readable label (e.g. `"Churn"`) |
| `column` | str | Source column name |
| `rate_pct` | float | Mean × 100 — the positive-class rate |

---

### `history_logs` entry
| Key | Type | Description |
|---|---|---|
| `time` | str | Timestamp `HH:MM:SS` |
| `msg` | str | Log message |
| `type` | str | `"system"`, `"info"`, or `"error"` |

---

### `analysis_results` entry
Accumulates after each completed research path

| Key | Type | Description |
|---|---|---|
| `path` | dict | The selected research path from Researcher Agent |
| `tool_result` | list[dict] | Raw Starter Kit outputs (one per tool instruction) |
| `da_findings` | dict | DA Agent interpretation — see `da_findings` below |

---

### `da_findings`
Produced by DA Agent — validated by Pydantic

| Key | Type | Description |
|---|---|---|
| `headline` | str | One-sentence finding summary |
| `key_insights` | list[str] | 3–5 bullet insights |
| `supporting_stats` | list[dict] | `{label, value, context}` per stat |
| `caveats` | str | Limitations or data quality warnings (may be empty) |
| `recommended_viz` | str | Suggested chart type |
| `viz_rationale` | str | One sentence explaining the chart choice |

---

### `chart_configs`
Produced by BI Developer Agent — validated by Pydantic

| Key | Type | Description |
|---|---|---|
| `dashboard_narrative` | str | One-paragraph executive framing |
| `kpis` | list[dict] | `{label, value, delta, context}` per KPI |
| `charts` | list[dict] | Plotly-compatible chart configs — see below |

**`charts` entry:**
| Key | Type | Description |
|---|---|---|
| `chart_type` | str | `"bar"`, `"line"`, `"scatter"`, or `"heatmap"` |
| `title` | str | Chart title |
| `x` | list | X-axis data |
| `y` | list | Y-axis data |
| `x_label` | str | X-axis label |
| `y_label` | str | Y-axis label |
| `source_path` | str | Research path title this chart derives from |
| `explanation` | str | Caption text explaining the chart |

---

### `synthesis`
Produced by Synthesis Agent — validated by Pydantic

| Key | Type | Description |
|---|---|---|
| `narrative` | str | Cross-path strategic narrative paragraph |
| `recommendations` | list[str] | Ordered actionable recommendations |
